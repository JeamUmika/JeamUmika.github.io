运算符 描述 例子
= 简单的赋值运算符 c = a + b
+= 加法赋值运算符 c += a 等效于 c = c + a
-= 减法赋值运算符 c -= a 等效于 c = c - a
*= 乘法赋值运算符 c *= a 等效于 c = c * a
/= 除法赋值运算符 c /= a 等效于 c = c / a
%= 取模赋值运算符 c %= a 等效于 c = c % a
**= 幂赋值运算符 c **= a 等效于 c = c ** a
//= 取整除赋值运算符 c //= a 等效于 c = c // a

成员运算符

运算符 描述 例子
in 如果在指定的序列中找到值返回 True，否则返回 False。 a = ‘ab’ in ‘abcd’ # True
not in 如果在指定的序列中没有找到值返回 True，否则返回 False。 a = ‘ab’ not in ‘abcd’ # False
